== Description ==

Simple widget for displaying current bitcoin quotes to currencies.

Current options of widget:

*   Fiat Currencies Support: USD, RUR, EUR, UAH.
*   May displaying several widgets with same page

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Appearance->Widgets screen to choose and configure the widget
